Добрый день!
Чтобы проверить ваши навыки, мы просим вас сделать простой плагин для WordPress согласно требованиям ниже.

Описание
Плагин с рабочим названием Employee info должен позволять управлять через админку списками сотрудников и выводить их на страницу с использованием шорткода.
Плагин должен корректно инсталлироваться и удаляться.


Backend
•	Информация о сотруднике должна быть сделана как новый тип постов и включать следующие поля:
o	Имя, фамилия;
o	Должность;
o	Фотография (аватар);
o	Чекбокс - сотрудник доступен или нет;
o	Пол (Male, Female, Unknown).



Frontend
•	Информация о сотруднике должна выводиться при помощи шорткода.
•	Блок карточки сотрудника должен быть responsive:
o	Для широких экранов фото на карточке сотрудника должно быть слева от текста;
o	При сужении экрана, текст должен переходить под фото;
•	Фон карточки сотрудника должен отличаться в зависимости от заданного пола.



Логирование
•	Для логирования информации надо создать отдельную таблицу в базе. Таблица должна иметь foreign key на сотрудников.
•	Когда шорткод показывается на странице сайта, логировать в отдельную табличку в базе: 
o	Дату;
o	Страницу (желательно);
o	Карточка какого сотрудника была показана.

Дополнительные требования
•	Не пользоваться вспомогательными плагинами вроде Custom Post Type UI.
•	Использовать Git в процессе разработки.
